Mime - Capturing Mock Library for Node.js

Mime is a capturing mock library for Node.js. It uses harmony-reflect Proxy objects (part of the ES 6 JavaScript standard) to allow for very simple capturing mock objects and capturing callbacks to be created and used within an automated test framework such as Mocha. Take a look at the Mime home page, the Mime API documentation or the Mime examples.
